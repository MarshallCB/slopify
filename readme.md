# Slopify

Convert depth maps to slope (dudv) maps

## Installation

```bash
  npm i -g slopify
```

## Usage

```bash
  slopify [input] [output]
```

```bash
  slopify --help
```

## Example

```bash
  slopify example/hill.png example/hill-terrain.png
```
